Shows next and previous woocommerce product in single product view.

== Description ==

Show the previous and next product on the single product view of your store, from the same category, on the Top the page.

== Installation ==

1. Install the 'Next Previous Button Woocommerce Single Product' plugin either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.
2. Activate the 'Next Previous Button Woocommerce Single Product' WordPress plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Do i need Woocommerce install in my site? =

Of course, Our plugin works with Woocommerce so you need Woocommerce shop in your site.
